Yukis Army knife について

概要

このソフトは Windows 上で使える多数の便利ツールを提供する、十徳ナイフのようなフトです。

コンセプトとして「簡単なことを簡単に実行する」を意識して作りました。

具体的には

  • コピー&ペースト支援や様々なテキスト変換を行える「テキスト・情報」

  • 簡易的な動画編集や変換が行える「動画・音声」

  • 画像編集や PDF 操作などが行える「画像・PDF」

  • WIndows システム上で行える様々な便利機能を使った支援や、ちょっとした便利ツールを使える「システム・小物」

の 4 分野に関わる 180 種類超のツールを使用できます。

具体的な使い方や機能については。こちらの wiki の方で確認をお願いします（導入方法の動画もあります）

Yukis Army knife wiki

ソフトの見た目


動作環境

Windows10 及び Windows11 で動作を確認しています。ただし WindowsAPI を多数使用しているため、仕様変更次第では一部機能が正常に動作しなくなるかもしれません。


開発

言語は主に Python を用いて開発されています。

GUI フレームワークには標準ライブラリである Tkinter を用いており、ビルドには Pyinstaller を使いました。

初めてのソフト制作だったので、有名で安定したライブラリを用いました。


注意事項

正直ソフト開発どころかプログラミングについてよく分からないまま見切り発車で制作したため、バグや不具合はかなりあります（見つけ次第修正していますが）

意図しない動作をしたら、まず一度ソフトの再起動を行い、それでも改善しない場合は X などで知らせてくれると助かります。


公開に関して

制作理由

もともと WIndows のフリーソフトを使って、パソコンの操作を効率化したり、できないことをできるようにすることが好きでした。

しかし便利なソフトが多すぎて、管理が煩雑になったり、導入したことを忘れたり、使い方を混乱したりすることが多かったです。

そこでよく使うツールを自分で実装し直して、一つのソフトにしてしまえば良いと思ったのがこのソフトを作るモチベーションとなりました。

実装時は一つのソフトに多機能を詰め込むのではなく、単機能のソフト収納する形で実装しているため、各種フリーソフトと同じように小回りのきくような使い勝手ができます。


感想

今まで統計分析などにしか Python を使っていなかったため、初のソフト開発は非常に得られる物が多かったです。

しかし知識・経験が皆無だったためコードが非常に雑なことや、見通しを立てずに見切り発車したためメガドライブタワーのように実装が積み重なっていることは非常に反省しています。

これからも、（一番の利用者である自分のためにも）今後も改善・機能追加していこうと思います。
