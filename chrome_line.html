PC ブラウザから LINE を開く方法（便利な裏技も紹介）

Posted on 2024年02月25日 (日) in ツール紹介

始めに

LINE をパソコン内にインストールしなくても、Chrome 拡張機能を使えばブラウザから LINE を見れることはご存知でしょうか？

わざわざ PC 版 LUNE を使わずに、Chrome 拡張機能版の LINE を使うメリットは以下のとおりです。

  • 導入が簡単
  • ブラウザを開いて拡張機能をインストールするだけ
  • PC 版と比べて非常に軽快に動作する
  • PC 版と異なり閉じたあとにタスクトレイに残らないため気軽

基本スマホで LINE をして、テキストやファイルを送りたい時ぐらいしか PC で LINE を使わない人にとっては、PC 版 LINE は非常に使いづらいです。

そのような人にとっては、この Chrome 拡張機能版の LINE は便利に使えると思います。

しかし Chrome 拡張機能版の LINE にもデメリットがあり、

  • 通話や投票などの機能は使えない

  • 昔のトークは見れないことがある

という欠点があるため、これらで困るときは大人しく PC 版 LINE を導入しましよう。

Chrome 拡張機能版の LINE の導入の仕方

  1. まずはこちらのサイトから Chrome 拡張機能をインストールします。

  2. 次に拡張機能のアイコンをクリックします。

  3. LINE のマークが表示されるため、それを押すと別のウィンドウから LINE が立ち上がります。

あとはログインすれば、いつでも LINE を開くことができます。

裏技紹介

Chrome 拡張機能版の LINE を開くと、別ウィンドウから立ち上がります。

この仕様に関して不便に思わない人は良いですが、普通にブラウザ内でみたい人もいると思います。（私がそうでした）

そこでブラウザ（のウィンドウ）内で LINE を開く裏技を紹介します。

  1. まず普通に LINE 拡張機能のボタンを教えて LINE を開き、ログインします。

  2. ログインしたら適当な場所で右クリックをし、下にある「ページのソースを表示」を押します。

  3. そうしたらブラウザにコードのようなものが表示されますが、それを無視して上の URL から、先頭の「view-source:」を除いた部分をコピーして、コピーした部分をブックマークに登録します。

  4. そうすればそのブックマークを押すと、ブラウザ内で LINE を開くことができます。

このようなやり方で LINE をブラウザ内で表示することが可能です。

ただしログインのページ内で「ページのソースを表示」を押しても上手くいかないため、必ずログインしたあとに右クリックの中にある「ページのソースを表示」を押してください。
