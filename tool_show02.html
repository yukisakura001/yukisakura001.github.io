DeepL
(万人におすすめ)

ダウンロード：https://www.deepl.com/ja/app/

高精度な翻訳ソフト。

web サイト版は有名だが、PC 版は便利なのにもかかわらずあまり有名でないため紹介。

タスクトレイ（右下の時計とかの左隣）に収納され、アイコンをクリック or ctrl+c×2 で表示できる。

web サイト版と同じように翻訳画面から翻訳もできるが、PC 版では「選択中のテキストを翻訳」「自分が書いているテキストを直接翻訳」「画像などテキストを選択できなくても範囲選択で翻訳」が可能で、どれも日頃の翻訳を便利に行うことができる。

またウィンドウ呼び出しや上記の翻訳方法のショートカットキーは設定から選択できるため、他と被った場合は変更が可能。

LhaForge
(こだわる人向け)

ダウンロード：泥巣 : 作品庫 - LhaForge

ファイルの解凍圧縮ソフト。

この手のソフトはたくさんあり最近は windows デフォルトでも結構困らないが、私はこのソフトを愛用している。

その理由は、「アイコンの分かりやすさ」「設定の豊富さ」が優れているからである。

このソフトのデフォルトのアイコンは別に普通だが、設定で拡張子ごとにアイコンを変更できるオプションがあり、それをオンにすると拡張子ごとに分かりやすいアイコンに変化する。

また設定画面を見れば分かるように、解凍・圧縮に関してのオプションが豊富であり、一度設定すれば他のソフトよりも使いやすいため非常に快適に使える。

圧縮の設定に関しては 7zip に劣り、解凍の精度に関しては Explzh に劣るため、これらと併用することをおすすめする。

HoneyView
(万人におすすめ)

ダウンロード：高速画像ビューアー Honeyview · ダウンロード/ 特徴

非常に軽量な画像ビューアー。

他の画像ビューワーソフトに比べ、モダンなデザインでありながら軽快に動作するため非常に取り回しが良い。

また私は使わないが簡単な画像の編集も可能である。

全体として高性能というわけではないが、細かい使い勝手や体験が素晴らしいのでおすすめである。

VLC
(万人におすすめ)

ダウンロード：オープンソースのベストなプレイヤー VLC メディアプレイヤーのオフィシャルダウンロードです。 - VideoLAN

ほぼすべてのメディアファイルを再生できる動画プレイヤーソフト。

コーデックや拡張子など難しいことを考えなくとも再生できるのが強み。

またツールバーのカスタマイズも意外と細かく設定することが可能で、自分にあったカスタマイズができる。

似たような動画プレイヤーはいくつもあるが、ツールバーのカスタマイズ性の良さから私はこのソフトを使っている。

Clibor
(万人におすすめ)

ダウンロード：ダウンロード | Clibor

クリップボード（コピペ）拡張ソフト。

起動しておくとコピーしたテキストが自動で保存され、ctrl×2 で履歴を表示し自由にペーストが可能。

また定型文もカテゴリごとに分けて登録ができるため、パソコンの入力の手間が大幅に楽になる。

似たような機能として Windows デフォルトの「クリップボードの履歴(Win+V)」機能もあるが、そちらに比べて一覧性の良さ・カテゴリ分けできる定型文・カスタマイズ性が遥かに勝っている。

カスタマイズ項目が非常に豊富で見た目も変えられるため、自分にあった使い方ができる。

使い道の多さ・利便性の高さから万人におすすめできるソフト。

終わりに

今回は誰が導入しても腐りにくいソフトを中心に紹介した。

それぞれ Windows のデフォルト機能で代替も可能だが、Windows 標準ソフトは絶妙に使いづらいため、お気に入りのソフトを探して使ったほうが良い。

ただ「クリップボード履歴」など Windows の便利なデフォルト機能についても理解しておくと、出先でも使えるため覚えておいても損はないと思われる。
