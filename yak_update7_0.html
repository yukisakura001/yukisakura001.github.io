変更点

ランチャーモード追加

概要

今回のメジャーアップデートの大目玉となるモードです。

「Shift+pause(fn+左 shift)」でランチャーを呼び出すことができます。

ランチャーにはボタンが機械的に並んでおり、そのボタンに

  • Yukis Army knife の各ツール

  • web サイト

  • ファイル・フォルダ

を登録することで、ツールの起動や web サイトの起動などができます。

ランチャー登録方法

これらのツールは未登録のボタンを左クリックすると出てくる登録画面を通じて登録ができます。

Yukis Army knife ツールの場合はツールを選択、web サイトの場合 URL、ファイル・フォルダの場合パスを入力することで登録がでる他、ファイル・フォルダに関してはドラッグ・アンド・ドロップで登録することもできます。

また登録済みボタンを左クリックすると「削除」「名前変更」が出てくるので、そちらで登録の削除・名前の変更を行うことができます。

その他

詳細設定からランチャーの行・列数を調整できるため、登録数に応じて使いやすいようにボタンの数を上下してください。

今後は

  • リストモードの実装

  • 全体的な操作感の改善とバグの修正

  • ショートカットによる登録の起動ができるように

などの改良を考えています。

タスクアイコン登録時に、入力した名前のツールがなければエラーを出すように

今まで間違えた名前を登録しようとすると、「アラート無しで登録されない」と言う処理を行っていたのですが、それだと分かりづらいという声があったためアラートを発するようになりました。

テキスト検索で正規表現が使えるように

テキスト検索機能で、正規表現に対応しました。

web ファイル圧縮 追加

web サイトに使う HTML・CSS・JavaScript をまとめて圧縮できる機能です。

画像圧縮系の機能と組み合わせて使えば、web サイトの大幅な軽量化を行うことができます。

ついでにこのサイトは Python 製の Pelican という SSG を使って作られていますが、Pelican には web ファイルをビルド時に軽量化する「pelican-minify」というプラグインがあるので、Pelican を使う方にはそちらがオススメです。

設定から必要ない機能を非表示にできるように

設定から「非表示にする機能を登録」を使えるようになりました。

この設定を使うと不要なツールをメイン画面から非表示にすることができます。

不要な機能が邪魔な時で必要な機能が探せないときのために作りましたが、ランチャーモードがこの後実装されたため半分存在意義がなくなりました。

zip 圧縮で単体ファイルも圧縮できるように

zip 圧縮機能でファイル単体でも圧縮できるようになりました。

逆にこの機能製作時になぜ単体圧縮をできないようにしたのか謎です。

パス付き zip 作成 追加

基本的に zip 圧縮機能と同じですが、こちらではパスワードを付けることができる機能です。

zip 圧縮機能にこの機能を組み込むことを考えたのですが、自分がたまに使う zip 圧縮機能に不具合が増えるとあると困るので分けて作ってあります。

この機能に問題がなければ、今後のアプデで統合されるかもしれません。

使い道で考えるとファイル暗号化機能と被りますが、Yukis Army knife を持っていないと複合できないあちらの機能よりも、こちらのほうが使いやすそうです。

ただ最近は PPAP のは利用が非推奨となっているので、こちらの機能もあまり意味ないかもしれません...

感想

今回は念願のメジャーアップデートです。

今までよく使う機能を設定するにはタスクトレイでの右クリックしかなく不便だったため、大幅にマシになったと思います。

最近は新たなツールの追加よりも既存ツールや全体的な使い勝手向上を心がけているため、機能数 200 突破は当分先になりそうです。
