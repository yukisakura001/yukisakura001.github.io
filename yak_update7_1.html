変更点

パスワード管理 追加

アカウントのログインに必要な「ユーザー名」「パスワード」「URL」などを保存できる機能です。

初回はパスワードを設定する必要があります。

保存したアカウントは右クリックからコピーや編集が可能である他、入力時に暗号化されて保存されているので安心して使うことができます。

この機能はまともなパスワードマネージャーが使えないときの代用品として作ったため、あまり性能は高くありませんが、ブラウザなどに登録したくないパスワードを管理するには使えるかもしれません。

ランチャーの関するバグをいくつか修正

ver7.0 で追加されたパスワードマネージャ機能ですが、使用に不便なバグがいくつかあったため修正しました。

地味ですが重要な修正が多かったので、地味に時間がかかってしまいました。

画像変換 から ico への変換を削除

画像変換機能から ico への変換を削除しました。

ico への変換は、新しく追加された「ico 作成」機能で行うことができます。

ico 作成 追加

画像ファイルを ico に変換できる機能です。

以前は「画像変換」で ico へのファイル変換ができましたが、実用する際にバグや不便な仕様が多く、より実用的にアイコン変換機能として使えるよう機能を独立しました。

そのため変換時にサイズ変換などもできるようになっています。

とりあえずでアイコンへの変換を行う際には、便利な機能になっていると思います。

web ブックマーク追加

これは URL を変換して、X などで見られる web サイトへのブックマーク？を HTML で作成する機能です。

まだ調整中で見た目もあんまり良くありませんが、今後少しづつ改善する予定です。

X などで URL を貼ると表示されるブックマークをブラウザで使うためにこの変換機能を作りましたが、代替ツールを見つけたらそっちに乗り換えてしまうかもしれません...

感想

今回は過去最小規模のアプデとなっています。

最近は、web 系の技術に手を出したり、映画にハマったり、資格勉強をはじめたり、仕事が忙しくなったりして全然このソフトのコードを書くことができませんでした...

しかしこのソフトの開発は、代替ソフトがどこかで生まれるまで続けるつもりです（というか私が一番使っているユーザー）。

これからもどうぞよろしくお願いします。
